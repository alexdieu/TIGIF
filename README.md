# TIGIF
A software to convert .mp4 , .mov and .mkv to .8xp 

Current status of the project : Needs optimizations, fix colors
Was before written in C using CE Toolchain. It's now
directly written in assembly for better and faster results !

Should be working on windows, macos and linux !
